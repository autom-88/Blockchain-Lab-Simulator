# ⛓️ Blockchain Lab Simulator

An interactive educational simulator built with **HTML**, **CSS**, and **JavaScript** to explore the core concepts of blockchain technology through realistic datasets and an intuitive dashboard.

The project is designed for students, developers, and cybersecurity enthusiasts who want to understand how a blockchain ecosystem works without connecting to a real network.

# 📚 Educational Modules

# ▶️ Running the Project

Because the simulator loads JSON files using the Fetch API, it **must** be served through a local web server.

### Visual Studio Code

Install **Live Server** and choose:

```
Open with Live Server
```

### Python

```bash
python -m http.server 8000
```

Open:

```
http://localhost:8000
```

### Node.js

```bash
npx http-server
```

or

```bash
npx serve
```
